# ClipMenuUtils
Custom actions for clip menu

## Setup
[Refer here](http://www.clipmenu.com/help/custom-actions/)

## Actions
### remove_brackets.js
This action removes the outer brackets from the clipText.
* (green apple shake) -> green apple shake
* [green apple shake] -> green apple shake
* {green apple shake} -> green apple shake
* (green apple shake] -> (green apple shake]
* green apple shake -> green apple shake
* {green apple shake -> {green apple shake
* green apple shake) -> green apple shake
